## âš™ï¸ How does the bidding system work?

Every time a user performs a search, an **automatic auction** is triggered among eligible ads.

Ad position is determined by the **ad-rank**, which considers:
- Ad quality (ad-score)
- Target ACOS
- Competition within the auction

Mercado Libre uses the **second best price** model, meaning:
- You do not pay your maximum bid
- You only pay what is necessary to beat the closest competitor

Some of the most important metrics available in the Campaign Manager are:

- **Impressions**: number of times the ad was shown
