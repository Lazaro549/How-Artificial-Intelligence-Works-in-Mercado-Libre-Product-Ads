# Product Ads Metrics Explained

This document explains the core metrics used by Mercado Libre Product Ads.

## Key Metrics

### Impressions
Number of times an ad is shown in search results.

### Clicks
Number of times users interact with the ad.

### CTR (Click Through Rate)
Clicks / Impressions.

### Conversion Rate
Sales / Clicks.

### Revenue
Total sales attributed to advertising.

### Ad Spend
Total amount invested in ads.

### ACOS
Ad Spend / Revenue.

ACOS is the main variable used by the bidding system to determine competitiveness.
